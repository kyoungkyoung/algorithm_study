1. 함수
    - 로직을 담아서 반복 사용
    - 함수 내부에 있는 변수는 Local(지역) 변수
    - 함수 외부에 있는 변수는 Global(전역) 변수

2. LEGB 규칙
    - local 영역에서 global 영역 변수를 재할당 할 수 없다.
        - 재할당하고 싶을 때에는 global 처리
    - local 영역에서 global 영역 변수를 참조는 할 수 있다.
    - 참조할 때에는 LEGB 순서로 참조한다
    - Local > Enclosed > Global > Built-in

3. 재귀함수
    - 자기자신으로써 정의되는 함수
    - 종료조건을 설정해주지 않으면 재귀함수의 depth가 너무 깊어지기 때문에 재귀에러가 발생한다.
    - 따라서, 재귀의 종료 조건을 작성해줘야함
    - 뒤에서부터 타고 올라가는 형식이다.
    - 항상 return 조건 (종료조건)을 먼저 작성해준다.

4. 이진탐색
    - 정렬된 리스트에서만 사용 가능
    - 범위가 정해져 있는 문제에서 사용 가능
    - 같은 리스트라도 정렬이 되지 않은 리스트는 이진탐색 사용 불가능
    - 예) 업다운게임
